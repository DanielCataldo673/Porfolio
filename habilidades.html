Habilidades

HTML 5

¿Qué es HTML5?

HTML es un lenguaje de marcado para crear páginas web. Utiliza etiquetas para estructurar y dar formato al contenido.

CSS 3

¿Qué significa CSS3?

CSS es un lenguaje de estilos utilizado para dar formato y diseño a páginas web creadas con HTML.

Bootstrap

¿Qué es Bootstrap y para qué sirve?

Bootstrap es un marco de trabajo de código abierto que facilita el diseño web responsivo y la creación de interfaces.

PHP

¿Qué es PHP?

PHP es un lenguaje de programación de código abierto, ampliamente utilizado para el desarrollo web y la creación de aplicaciones.

MYSQL

¿Qué es MySQL?

MySQL es un sistema de gestión de bases de datos relacionales de código abierto, ampliamente utilizado en aplicaciones web.

GitHub

¿Qué es GitHub?

GitHub es una plataforma de alojamiento de código fuente y colaboración para proyectos de software, basada en Git.

Testing QA Manual

¿Qué es Testing QA Manual?

El Testing QA Manual es un proceso de verificación y validación de software realizado manualmente por un equipo especializado.

Metodologia Agil-Scrum

¿Qué es la Metodologia Agil-Scrum?

Scrum es una metodología ágil de gestión de proyectos que se centra en la colaboración, la flexibilidad y la entrega incremental.

Habilidades Blandas

¿Qué son las Habilidades Blandas?

Las habilidades blandas son atributos personales como la comunicación, el trabajo en equipo, la resolución de problemas y la empatía.
